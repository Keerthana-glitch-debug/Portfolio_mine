Dam Monitoring Using Remote Sensing

A remote sensing project that analyses dam structural integrity and reservoir stability using satellite imagery and geospatial data. Utilises SAR (Synthetic Aperture Radar) backscatter analysis via Google Earth Engine and Sentinel-1 data to detect temporal changes in water extent and surface deformation. Outputs include slope stability maps, risk zone classification layers, and change detection GeoTIFFs — visualised in QGIS for spatial analysis and reporting.
